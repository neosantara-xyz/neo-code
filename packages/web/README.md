# @neosantara-xyz/web

Landing page for Neo Code built with Next.js and NES.css 8-bit aesthetic.

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

Produces a static export in the `out/` directory.

## Deploy to Cloudflare Pages

```bash
npm run deploy
```

Deploys the `out/` directory to Cloudflare Pages via Wrangler.
